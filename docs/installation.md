# Installation

## Stable release

To install toolbox, run this command in your terminal:

```sh
uv add toolbox
```

Or if you prefer to use `pip`:

```sh
pip install toolbox
```

## From source

The source files for toolbox can be downloaded from the [Github repo](https://github.com/binnietom/toolbox).

You can either clone the public repository:

```sh
git clone https://github.com/binnietom/toolbox
```

Or download the [tarball](https://github.com/binnietom/toolbox/tarball/main):

```sh
curl -OJL https://github.com/binnietom/toolbox/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd toolbox
uv sync
```
